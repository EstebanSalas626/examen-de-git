## Requisitos

- Node.js (recomendado: versión LTS actual)
- npm o yarn

## Instalación

```bash
npm install
```

o si usas yarn:

```bash
yarn install
```

## Uso

Actualizar con el comando principal de tu proyecto. Por ejemplo:

```bash
npm start
```

o:

```bash
yarn start
```

## Scripts comunes

- `npm start` / `yarn start`: Inicia la aplicación
- `npm test` / `yarn test`: Ejecuta las pruebas
- `npm run build` / `yarn build`: Genera los archivos de producción

## Estructura sugerida

- `src/` - Código fuente de la aplicación
- `tests/` - Pruebas unitarias e integración
- `package.json` - Dependencias y scripts
